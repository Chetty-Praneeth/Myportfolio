DigitRecog

Digit Recognition on MNIST Dataset

About the Project

DigitRecog is a machine learning model developed to recognize handwritten digits using the popular MNIST dataset. This project utilizes a convolutional neural network (CNN) trained on 60,000 samples of handwritten numbers, achieving a high accuracy rate in identifying digits from 0 to 9. It's a great demonstration of deep learning applied to image classification.
